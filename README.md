# Bank-Loan-Risk-Credit_Exposure-Analysis-Dashboard-

📊 Bank Loan Risk Analysis Dashboard (Built in Excel) In this project, I analyzed a financial loan dataset to identify high-risk applicants and uncover credit behavior patterns.  

Analyzed 500+ loan records with total exposure of ₹526M+ to evaluate credit risk and portfolio performance. Identified a 17.2% default rate through analysis of credit scores (avg. 569) and interest rates (avg. 10.89%). Developed interactive dashboards to monitor loan distribution, risk segmentation, and default trends, enabling data-driven decision-making for credit risk assessment.
